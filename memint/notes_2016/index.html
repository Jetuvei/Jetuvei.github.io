Membrane Interface Model

Below are links to a set of documents explaining the Membrane-Interface model.

  • I recommend you start with "MI_overview".
  • The document "general_MF" contains some useful progress on the general mean-field theory.
  • In "mf_summary" are the details of the p=1 mean-field theory calculation, including the transfer matrix/Airy function results suggesting a width exponent of 1/3.
  • The document "mf" is quite poorly presented, and a lot of the content can be found in either "mf_summary" or chapter 5 of my thesis. This document is however the only place I really go into the explicit leading order dependence on the system size L of a lot of the significant quantities.

All the documents should have a date at the top, which will be updated if I make a newer version.
